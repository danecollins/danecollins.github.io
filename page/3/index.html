Portland #36

Oct 7, 2010 2 min.

One of the first things we noticed walking by Whales off the Coast of Maine was the sea turtle which we saw for the first time. This wall is in a great spot as you can walk right along it for the full length of the pier. The sun has taken it’s toll on this wall with the images showing significant fading but it’s definitely worth seeing. Like the wall in New Bedford this wall let’s you see several different types of whales as well as dolphins all on one wall.
